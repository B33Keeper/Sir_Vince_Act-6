# Movie Review UI - Frontend

React frontend application for the Movie Review system.

## Installation

```bash
npm install
```

## Running the app

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

## Features

- View all movies with average ratings
- Add, edit, and delete movies
- Add reviews with 1-5 star ratings
- View all reviews for each movie
- Responsive design

## API Configuration

Make sure the backend server is running on `http://localhost:3001`. The API base URL is configured in `src/services/movieService.js` and `src/services/reviewService.js`.
